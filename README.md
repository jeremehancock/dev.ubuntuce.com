## Usage

```bash
curl -s --compressed "https://dev.ubuntuce.com/KEY.gpg" | sudo apt-key add -
sudo curl -s --compressed -o /etc/apt/sources.list.d/ubuntuce-dev.list "https://dev.ubuntuce.com/ubuntuce-dev.list"
sudo apt update
```
